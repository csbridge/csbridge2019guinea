Lavez 100 Pièce des Habilles (Place 100)

Problème écrit par Chris. Adapter par Lisa E.

Documents à distribuer: KarelReference

Votre objectif est de commisioner Karel à laver 100 pièce des habilles (signifier par les bips). S'il vous plaît!!! Ne tapez pas putBeeper() 100 fois de suite! (indice: utilisez une for loop.)

Solution

/**
 * Programme: Lavez 100 Pièce des Habilles (Place100)
 * -------------
 * Ce programme permet à Karel de lavez une pile de 100 pièce des habilles.
 */

public class Place100 extends SuperKarel {

	public void run() {
		move();
		// Cette for loop répétera le code 100 fois.
		for(int i = 0; i < 100; i++) {
			putBeeper();
		}
		move();
	}
}
