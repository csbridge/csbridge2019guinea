Placez 100 (avec la solution)

Problème écrit par Chris

Documents à distribuer: KarelReference

Votre objectif est que Karel crée une pile de 100 bips sonores sur la case devant laquelle elle commence. S'il vous plaît!!! Ne tapez pas putBeeper 100 fois de suite! (indice: utilisez une boucle for.)

Solution

/**
 * Programme: Place100
 * -------------
 * Ce programme permet à Karel de placer une pile de 100 signaux sonores. Bon temps.
 */
public class Place100 extends SuperKarel {

	public void run() {
		move();
		// Cette boucle for répétera le code 100 fois.
		for(int i = 0; i < 100; i++) {
			putBeeper();
		}
		move();
	}
}
