3. Nombres Aléatoires

Code: NombresAleatoires.java

Ecrivez un programme qui affiche 1000 nombres aléatoires entre 0 et 100

Le fichier NombresAleatoires.java a un code qui génère un seul nombre aléatoire. 

public class NombresAleatoires extends ConsoleProgram {
							   
	// Un generateur de nombres aleatoires
	private RandomGenerator rg = new RandomGenerator();

	public void run() {
		// change ce code pour afficher 1000 nombres aleatoires
		// entre 0 et 100
		int nombreAleatoire = rg.nextInt(0, 10);
		println(nombreAleatoire);	
	}
}

Pour pouvoir générer des nombres aléatoires, créez une variable qui est une instance de RandomGenerator (la variable est déclarée en dehors des autres méthodes) comme suit :

private RandomGenerator rgen = new RandomGenerator();

Vous pouvez ensuite utiliser cette variable pour générer des nombres aléatoires.

rgen.nextInt(min, max) // génère un int aléatoire dans l'intervale (min, max) inclus
