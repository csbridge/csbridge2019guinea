3. Nombres Aléatoires

Ecrivez un programme qui imprime 1 000 nombres aléatoires compris entre 0 et 100.

Le fichier RandomNumbers.java a un code qui génère un seul nombre aléatoire. 

public class RandomNumbers extends ConsoleProgram {

   // Un générateur de nombres aléatoires
   private RandomGenerator rgen = new RandomGenerator();

   public void run() {
      // changer ce code pour imprimer 1000 nombres aléatoires
      // dans la plage de 0 à 100.
      int example = rgen.nextInt(0, 10);
      println(example);
   }
}

Afin de générer des nombres aléatoires, créez une variable d'instance RandomGenerator (une variable déclarée en dehors de toutes les méthodes) comme ceci:

private RandomGenerator rgen = new RandomGenerator();

Vous pouvez ensuite utiliser la variable pour générer des nombres aléatoires.

rgen.nextInt(min, max) //génère un int aléatoire dans la plage (min, max) inclus
