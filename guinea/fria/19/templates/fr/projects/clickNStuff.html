Référence d'événements

Ceci est un programme de bonus! C'est un peu difficile, mais terminer ce programme vous aidera certainement à écrire votre projet final.

Écrivez GraphicsPrograms qui font une animation après que l'utilisateur clique dessus.

Première partie: Mettre l'alphabet anglais en pause

In ClickNPause.java, écrivez un GraphicsProgram qui fait une boucle dans l’alphabet anglais (de A à Z). Ajoutez également des fonctionnalités de sorte que lorsque l'utilisateur clique quelque part sur l'écran, un libellé "Pause ..." s'affiche à l'endroit où l'utilisateur a cliqué.

Conditions générales d'Utilisation:

  • Utilisez un délai de 100 millisecondes ou moins entre chaque caractère alphabétique.
  • Ajoutez une pause de 1 seconde après chaque clic de l'utilisateur.
  • Commencez par des nombres animés avant l’alphabet pour établir votre boucle d’animation principale.
  • Ajouter un pausing... étiquette qui montre où l'utilisateur a cliqué.

Animer les personnages

Puisque nous animons des lettres et non des entiers, nous devrions garder une trace d'un char variable de classe. une char est un seul caractère long (donc différent de String, qui peut avoir plusieurs caractères). Pour définir le texte d'un GLabel sur un seul caractère:

char c = 'c';
label.setLabel(c + "");

L'avantage d'utiliser un char sur une chaîne est que chaque caractère a une représentation numérique unique, en utilisant quelque chose que nous appelons ASCII. Donc si on convertissait le personnage A en un entier:

System.out.println((int) 'A'); // renvoie 65

De même, la représentation numérique de Z is 65 + 26 - 1 = 90. La char à int la cartographie est décidée par le ASCII Codes.

Utilisez cet aperçu pour concevoir une méthode permettant de parcourir les lettres de A à Z, puis de revenir à A de nouveau.

Event Flags

Ajoutez un écouteur de souris de sorte que, lorsqu'un utilisateur clique, le compteur arrête temporairement d'incrémenter et que le signe "pause ..." s'affiche à l'emplacement sur lequel l'utilisateur a cliqué. Après une seconde, le signe "pause ..." devrait disparaître et le compteur continuer à augmenter.


Il sera difficile de mettre en œuvre tout cela en mouseClicked()!!! Vous ne pouvez pas faire de pause à l'intérieur d'une méthode d'écoute telle que mouseClicked() ou mouseMoved(), parce que ces méthodes sont conçues pour agir rapidement. Au lieu de cela, vous devez utiliser Event Flags.

une flag est une variable booléenne définie par le programmeur true quand un événement se produit, et est remis à false quand l'événement se termine. En pratique, un programmeur attribue la valeur true aux indicateurs d'événement dans les écouteurs de souris, vérifie s'ils sont vrais dans la boucle d'animation principale, puis effectue une action après avoir constaté que l'indicateur est activé. Voici un exemple de boucle d'animation principale qui vérifie également un indicateur:

private boolean eventFlag = false;
public void run() {
    while(true) {
        if (eventFlag) {
            doSomethingSpecial();
            eventFlag = false; // réinitialiser le drapeau une fois que vous avez terminé
        }
        doSomethingNormal();
        pause(DELAY);
    }
}

public void mouseMoved(MouseEvent e) {
    eventFlag = true; // allumer le drapeau
}

Avec cette nouvelle connaissance, terminez la première partie.

Deuxième partie: Le saut incroyable Karel

In ClickNJump.java, mettre en œuvre un programme où Karel (ci-dessous) se déplace pour toujours. Lorsque l'utilisateur clique dessus, Karel saute haut, atterrit et continue de se déplacer d'un côté à l'autre.

Conditions générales d'Utilisation:

  • Sauver le png Karel à la src dossier où se trouvent vos fichiers java. Vous pouvez trouver l'emplacement de votre projet en allant sur Eclipse, en cliquant avec le bouton droit de la souris sur le dossier du projet et en sélectionnant "Propriétés".
  • Actualisez le dossier dans Eclipse jusqu'à ce que karel.png visible dans l'explorateur de projet.
  • Karel devrait se déplacer en douceur, donc environ une fois toutes les 50 millisecondes (ou même plus fréquemment).

Lignes directrices pour le saut d'obstacles:

  • Le saut total devrait durer deux secondes.
  • Pendant que Karel saute, elle ne devrait pas pouvoir recommencer le saut.
  • Commencez par faire sauter Karel verticalement de haut en bas, à vitesse constante.
  • Ensuite, pensez à ajouter en gravité. Quelle devrait être votre vitesse de départ?
  • Enfin, ajoutez un mouvement horizontal pendant que Karel saute.
