1. Mad Max

Complétez le MadMax programme en remplissant le max méthode qui renvoie le plus grand de ses deux paramètres:

private int max(int i, int j)

La méthode run a déjà été écrite: 

public void run() {
    println("The max of 4 and 5 is: " + max(4, 5));
    println("The max of -4 and 4 is: " + max(-4, 4));
    println("The max of 1 and 2 is: " + max(1, 2));
    println("The max of 0 and 0 is: " + max(0, 0));
}

2. dessiner un arbre

Remplissez le drawTree méthode qui dessine un arbre avec le base de son tronc centré sur la position (x, y) prévue:

private void drawTree(int x, int y)

La méthode run a déjà été écrite: 

public void run() {
    drawTree(50, 150);
    drawTree(100, 300);
    drawTree(500, 200);
    drawTree(300, 250);
    drawTree(700, 310);
}

Chaque arbre doit avoir un camion d'une largeur de 20 pixels et d'une hauteur de 80 pixels. Les feuilles doivent avoir un rayon de 30 pixels. Votre sortie devrait ressembler à ceci:
