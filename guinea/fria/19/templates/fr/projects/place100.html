Placer 100 (avec solution)

Problem écrit par Chris

Référence: Karel Reference
Code: Placer100.java (Jour1)

Votre objectif est de faire que Karel dépose 100 beepers sur la cellule en face de sa position de depart. Ne tapez surtout pas putBeeper(); 100 fois! (Astuce: utiliser une boucle "pour")

Avant

Après

Solution

/**
 * Programme: Placer100
 * -------------
 * Ce programme fait que Karel depose une pile de 100 beepers.
 */

public class Place100 extends SuperKarel {
	
	public void run() {
		move();

		// Cette boucle "pour" ("for" loop) répétera le code dans son corps 100 fois.
		for(int i = 0; i < 100; i++) {
			putBeeper();
		}

		move();
	}
}
