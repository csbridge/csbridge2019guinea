Reference des programmes Console

Ecrit par Danielle Kain, Kevin Miller

Les programmes Console permettent les interactions and votre programme et l'utilisatuer. Pour écrire des programme console, assurez-vous que votre classe hérite de la classe ConsoleProgram: 
public class NomDeMaClasse extends ConsoleProgram {
    // ...
}

Les programme consoles peuvent peuvent recevoir des informations de l'utilisateur. Voici quelques exemples: 
readLine(String messageDinvite); // Lis et retourne une chaine de caracteres saisit par l'utilisateur
readInt(String messageDinvite); // Lis et retourne un nombre entier saisit par l'utilisageur
readDouble(String messageDinvite); // Lis et retourne un nombre reel saisit par l'utilisateur
println(String messageAImprimer); // Imprime un message sur la console
