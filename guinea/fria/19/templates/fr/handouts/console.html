Référence des programmes console

Ecrit par Danielle Kain, Kevin Miller. Traduit par Moussa Doumbouya.

Les programmes consoles permettent des interactions avec l'utilisatuer. Pour écrire des programmes console, assurez-vous que votre classe hérite de la classe ConsoleProgram: 
public class NomDeMaClasse extends ConsoleProgram {
    // ...
}

Les programmes consoles peuvent afficher des informations pour l'utilisateur. 
println(String messageAImprimer); // Imprime un message sur la console

Les programmes consoles peuvent recevoir des informations de l'utilisateur. Les appels de methodes suivantes lisent et retournent des informations de type différent saisits par l'utilisateur 
readLine(String messageInvite); // une chaine de caractères
readInt(String messageInvite); // un nombre entier
readDouble(String messageInvite); // un nombre réel
Le paramètre messageInvite ci-dessus est le message que votre programme affiche quand il invite l'utilisateur a saisir et entrer une information. %include templates/parts/footer.html
