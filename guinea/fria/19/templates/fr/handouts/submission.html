Submitting Assignments

At the end of each day, before you go home, submit your work!

How to Submit

To submit follow these three steps:

  1. Zip the folder containing your project. A helpful tutorial video can be found here.
  2. Name the Zip using your full name and the project name. For example I would call my zip: Tyler_Conklin_Random_Painter.zip if I was submitting the 'Random Painter' project.
  3. Upload the file using the link your section leaders in lab provided for you.
