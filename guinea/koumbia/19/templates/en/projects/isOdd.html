This demo shows a simple method that takes in input and returns whether or not that input is odd.

Solution

public class IsOdd extends ConsoleProgram {

	public void run() {
		for(int i = 0; i < 100; i++) {
			println(isOdd(0));
		}
	}

	private boolean isOdd(int value) {
		return (value % 2) == 1;
	}
}
