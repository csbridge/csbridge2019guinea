Place 100 (with solution)

Problem written by Chris. Adapted by Lisa E.

Handouts: KarelReference

Your goal is to have Karel wash 100 pieces of clothing (signified by beepers). Please!!! Do not type out putBeeper 100 times in a row! (hint: use a for loop.)

Solution

/**
 * Program: Wash 100 pieces of Clothing (Place100)
 * -------------
 * This program makes Karel wash a pile of 100 pieces of clothing. 
 */

public class Place100 extends SuperKarel {
	
	public void run() {
		move();
		// This for-loop will repeat the code inside 100 times.
		for(int i = 0; i < 100; i++) {
			putBeeper();
		}
		move();
	}
}
